 # recetas-cli

  Proyecto formativo desarrollado con Spring Boot, Spring MVC y Thymeleaf.

  ## Descripción
  Aplicación web cliente que consume la API REST `recetas-api` para gestionar recetas, dificultades y usuarios.

  ## Tecnologías utilizadas
  - Java 17
  - Spring Boot
  - Spring MVC
  - Thymeleaf
  - RestTemplate
  - Maven

  ## Funcionalidades
  - Inicio de sesión de usuario
  - Listado y gestión de recetas
  - Listado y gestión de dificultades
  - Administración básica de usuarios
  - Consumo de una API REST externa

  ## Proyecto relacionado
  Backend del proyecto:
  [recetas-api](https://github.com/j-alberto-lopez/recetas-api)

  ## Nota
  Este proyecto tiene finalidad formativa y académica.
